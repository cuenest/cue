# @cue/desktop

**Windows / macOS / Linux** app. Capacitor (or Tauri) wraps the same web UI in a native desktop shell, with a global capture hotkey and tray support.

Imports `@cue/engine` + `@cue/ui`. A desktop left always-on is the natural "always-on hub" for a household.

Status: not started (post Phase 0 web).
