# @cue/hub

**Headless self-hostable sync node.** No UI — runs on a VPS, Raspberry Pi, or always-on desktop as the household's always-on rendezvous + full backup replica.

Relays **encrypted** Yjs diffs between devices (it cannot read them) and keeps a durable replica so devices sync even when none of the personal devices are both online at once.

This is the OSS, zero-cost alternative to the paid `cue-cloud` relay — and a favorite for the self-hosting / homelab community.

Status: Phase 1.
